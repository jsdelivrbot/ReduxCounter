### HTML Tables

An HTML table comprises three parts:
1. Header   <thead>
2. Body     <tbody>
3. Footer   <tfoot>

Data in the table body (<tbody>) is grouped in columns and rows:
1. Header   <th>    the column name
2. Row      <tr>    a row
2. Cell     <td>    individual data cells
